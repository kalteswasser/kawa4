While desktop is still the preferred device for online shopping, mobile is becoming more and more important if you want to run a successful e-commerce.

Responsive design is no longer enough. You need to have a specific mobile SleekBoxes if you want to convert mobile users without being punished by Google.

On January 10, 2017, a new Google update went into effect that punishes mobile sites that use intrusive interstitials.

This could be a popup that covers the main content that a user has to dismiss before being able to see the content of the website.

To ensure a mobile-friendly user experience, we’ve created a mobile editor for your SleekBoxes.

This means you can create custom mobile SleekBoxes for mobile only that ensures you’ll never be punished by Google for having intrusive interstitials on your mobile site.

How Does the Mobile Editor Work?

With mobile SleekBoxes, you can now target mobile users specifically for more conversions.

And even better:

You can create any type of mobile SleekBox without having to worry about it being intrusive—it’s simply not possible.

All mobile SleekBoxes will show as a teaser in the bottom of the mobile screen and are only triggered when a user clicks the teaser.

This means you can convert mobile users into quality leads and customers without hurting the mobile user experience.

Our SleekBoxes are conversion-ready for all mobile devices. In fact, you can even use our device-specific rule to create targeted on-page messages for mobile users only.
